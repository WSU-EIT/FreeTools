What is Mailman?

Mailman is the list management software that powers lists.wsu.edu. From these pages, you can request to add, remove, and manage any lists you own. Users can even request password help for their lists.

Email Lists at WSU

You can request and administer your Mailman lists using these pages. Please contact the ITS Help Desk for assistance.

Common List Tasks

Request a List

Request a new email mailing list. Lists may only be requested by staff and faculty members at WSU. Student group or organization lists must be sponsored by a staff or faculty member.

Manage a List

Control most functions of your list. Make changes to your list configuration, including privacy and archiving.

Moderate a List

For lists that require posting approval, owners and moderators can log into the moderation pages to allow messages.

Delete a List

Request that a list you own be deleted.

Join a List

View a list of public mailing lists. Become a member of a public list, or request access to a private list.

Leave a List

Unsubscribe from a mailing list you have previously joined.

Request my password (Subscribers Only)

Mailman automatically creates passwords for each list a user is subscribed to. If you need your password sent to you, please select this option.

View a List Archive

Mailman lists are archived by default to the web. Access the archive of these lists here.
